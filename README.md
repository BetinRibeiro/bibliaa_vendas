# 📚 A Bíblia de Vendas (Versão Dev)

> **"Sua atitude determina sua altitude."**
> Uma adaptação técnica da filosofia de Jeffrey Gitomer, curada e masterizada por **Rogoberto Ribeiro da Silva**.

Este projeto é uma implementação digital e estruturada dos princípios fundamentais de vendas, organizada como um sistema de aprendizado contínuo (**CI/CD Mental**). O conteúdo transforma conceitos de vendas em um "Kernel" sólido para profissionais que buscam performance em produção.

## 🚀 Live Demo
Acesse o projeto em: [https://betinribeiro.github.io/bibliaa_vendas/](https://betinribeiro.github.io/bibliaa_vendas/)

## 📂 Estrutura do Projeto
O site está organizado de forma sequencial para garantir a entrega de valor em cada etapa:

* **index.html**: Capítulo 1 - A Filosofia do Sucesso.
* **2.html a 11.html**: A Jornada Evolutiva do Vendedor.
* **12.html**: Atitude e Lucro (Manutenção do Sucesso).
* **ultimocapitulo.html**: Metodologia de Performance.
